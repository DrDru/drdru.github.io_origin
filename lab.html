Welcome to this part of the lab.
Here you will find experiments with a smaller scope than the Necodesca project.

Warning : DrDru's lab is a dangerous place. You enter it at your own risk.

  • Necodesca in English : in this project I tried to rewrite the site in English to see how well the concept can be used for other languages
  • Necodesca in French
  • Stories in the mas form
  • Necodesca practice : this is a Ruby app I made to complement the site. The idea is to make the user read proper Japanese sentences and have a quick feedback mechanism to check his understanding
  • Grammar practice : this is a fork of a Japanese Grammar App. I played with it to get some familiarity with Javascript
  • My FF7 site on neocities : in this project I created a website that uses screenshots and dialogues from FF7 to teach Japanese
  • Blog : I write a few blog posts from time to time.
  • Knight emoji proposal : a submission request sent to the unicode consortium to add the knight emoji to the standard emoji set. It's been rejected :-(
  • The tile world chronicles : visit the part of this site dedicated to The Tile World Chrnocles, a set of stories in a fantasy world written in easy Japanese.
  • Return to Parscythe : a really cool comic I have started translating into easy Japanese
